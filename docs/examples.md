# ZARA Language Examples

## Basic Operations
```zara
set x = 10
set y = 3
set result = x + y * 2
show result
```

## Comparisons and Conditionals
```zara
when result > 10:
    show "Result is bigger than 10"
```

## Looping
```zara
set counter = 1
loop 3:
    show "Iteration number"
    show counter
    set counter = counter + 1
```
